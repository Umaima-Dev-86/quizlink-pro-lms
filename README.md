# Catalog-Z Website Clone
This project is a pixel-perfect clone of the **Catalog-Z** template (TemplateMo 556), built as a coding challenge. The primary focus of this project was to implement a responsive layout using **HTML5** and **CSS3 (Flexbox)**.

## 🛠 Tech Stack
*   **HTML5** (Semantic structure)
*   **CSS3** (Flexbox for layout, custom styling)
*   **Responsive Design**

## 🎯 Key Features
*   Pixel-perfect replication of the original design.
*   Fully responsive layout implemented with Flexbox.
*   Accurate typography and spacing based on the reference template.
*   Clean, semantic HTML structure.

## 📁 Repository Structure
*   `/index.html` - Main structure of the landing page.
*   `/css/` - Contains the main stylesheet.
*   `/img/` - Contains all assets sourced from the reference website.

*Created as part of a front-end development task.*A premium Multi-Page LMS Ecosystem featuring interactive diagnostic assessment modules, real-time analytics tracking, and smart spaced-repetition flashcards across 6 core technical domains. Built with responsive soft-pastel UI architecture.
